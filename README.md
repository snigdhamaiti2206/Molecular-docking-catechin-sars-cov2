# Molecular Docking Analysis of Catechin Gallate with SARS-CoV-2 Main Protease (NSP5)

## Overview
This project focuses on molecular docking analysis of Catechin Gallate with SARS-CoV-2 main protease (NSP5) to evaluate binding affinity and interaction.

## Tools Used
- AutoDock / AutoDock Vina
- PyMOL
- Bioinformatics tools

## Methodology
- Protein preparation
- Ligand preparation
- Docking simulation
- Analysis of binding affinity

## Results
- Observed binding interactions between ligand and NSP5
- Evaluated docking scores and stability

- Project Report (PDF)
- Results images
